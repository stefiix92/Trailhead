# Branding / social preview

This repo includes two basic branding assets under [`docs/assets/`](assets/):

- `docs/assets/logo.png`
- `docs/assets/social-preview.png` (OpenGraph 1280×640)

## Set the GitHub social preview image

In GitHub repo settings:

1. Go to **Settings → General**
2. Under **Social preview**, upload `docs/assets/social-preview.png`

This controls how the repo looks when shared (HN/Reddit/X/etc).
